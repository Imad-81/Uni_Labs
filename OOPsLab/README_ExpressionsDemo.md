# 🎲 Expressions Demo

> Illustrates basic arithmetic and logical expressions in Java and outputs their results.

---

## 📌 Overview

`ExpressionsDemo.java` performs several simple calculations and boolean operations, then prints the outcomes. It demonstrates how Java evaluates:

* Integer arithmetic with parentheses and division.
* Floating-point multiplication using scientific notation.
* Boolean logic using `&&` (AND) and `||` (OR).

---

## 🔍 Concepts Covered

1. **Arithmetic Operators**: `+`, `/`, `*`.
2. **Operator Precedence and Parentheses**: Ensures correct evaluation order.
3. **Scientific Notation**: `3.0e-6` represents `3.0 × 10⁻⁶`.
4. **Boolean Logic**: Combining `true`/`false` values using logical operators.

---

## 🚀 Running the Program

1. Open a terminal and change to this directory:
   ```bash
   cd OOPsLab
   ```
2. Compile the source file:
   ```bash
   javac ExpressionsDemo.java
   ```
3. Execute the compiled class:
   ```bash
   java ExpressionsDemo
   ```

You should see printed lines showing each expression and its computed value.
