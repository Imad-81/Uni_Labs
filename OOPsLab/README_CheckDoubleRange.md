# 🔍 Double Range Checker

> Accepts two floating-point numbers and determines if both fall strictly between 0 and 1.

---

## 📌 Purpose

`CheckDoubleRange.java` uses the `Scanner` class to read two `double` values from the user. It evaluates whether each number lies in the open interval (0, 1). If both values satisfy the condition, the program prints `true`; otherwise it prints `false`.

Practices include:

* Reading `double` inputs from the console.
* Writing compound boolean expressions with `&&`.
* Understanding exclusive range comparisons (`>` and `<`).

---

## 📁 Sample Interaction

```text
Enter first double value: 0.5
Enter second double value: 0.9
true
```

---

## 🚀 Execution Steps

```bash
cd OOPsLab
javac CheckDoubleRange.java
java CheckDoubleRange
```

Provide two numbers when prompted to observe the output.
