# 🔢 Check Four Equality

> Reads four integers from the user and reports whether they are all equal.

---

## 📝 Description

The program defined in `CheckEqualFour.java` prompts the user to input four numbers one by one. It then compares them using chained equality checks (`==`) and prints `equal` if every value matches, otherwise it prints `not equal`.

This exercise reinforces:

* Using the `Scanner` class for console input.
* Basic control flow with `if`/`else`.
* Chaining relational operations.

---

## 📂 Example

```text
Enter first number: 5
Enter second number: 5
Enter third number: 5
Enter fourth number: 5
equal
```

---

## 🚀 How to Run

```bash
cd OOPsLab
javac CheckEqualFour.java
java CheckEqualFour
```

Provide four integer values when prompted to see the result.
